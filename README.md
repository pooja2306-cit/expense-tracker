**Household Expense & Savings Tracker**:

A modern React.js web application to track household expenses, calculate monthly savings, and visualize spending analytics using interactive charts.

**Features**:

✅ Add household expenses
✅ Delete expenses
✅ Monthly income tracking
✅ Savings calculation
✅ Category-wise filtering
✅ Pie chart analytics using Recharts
✅ Budget warning system
✅ Dark mode / Light mode
✅ Responsive modern UI
**Expense Categories**:
Rent
Electricity
Grocery
Food
Travel
Internet
Transport
Water Bill
Entertainment
Others
**Tech Stack**:
React.js
JavaScript
CSS3
Recharts
**Budget Warning System**:

The application automatically alerts users when:

Expenses exceed 80% of income ⚠️
Expenses exceed total income ❌
